# EXCEL2XMIND

将Excel格式的测试用例转换为XMind思维导图格式

## 功能特点

- 支持多层级结构（一级模块、二级模块、三级模块）
- 支持优先级标记（P0/P1/P2/P3），自动转换为对应颜色的图标
- 预期结果作为测试步骤的子节点展示
- 兼容XMind 8格式

## 环境要求

- Python 3.x
- pandas
- openpyxl

## 安装依赖

```bash
pip install pandas openpyxl
```

## Excel文件格式要求

Excel文件需要包含以下列（按顺序）：

| 列序号 | 列名 | 说明 |
|--------|------|------|
| 0 | 一级模块 | 顶层分类 |
| 1 | 二级模块 | 子分类 |
| 2 | 三级模块 | 具体场景 |
| 3 | 测试场景 | 测试点描述 |
| 4 | 前置条件 | 测试前置条件 |
| 5 | 测试步骤 | 具体操作步骤 |
| 6 | 预期结果 | 期望结果 |
| 7 | 优先级 | P0/P1/P2/P3 |

## 使用方法

```bash
# 转换默认文件
python excel_to_xmind.py

# 转换指定文件
python excel_to_xmind.py "路径\到\你的\文件.xlsx"

# 指定输出路径
python excel_to_xmind.py "输入文件.xlsx" "输出文件.xmind"
```

## 优先级标记

| 优先级 | 标记颜色 | 说明 |
|--------|----------|------|
| P0 | 红色 | 最高优先级 |
| P1 | 橙色 | 高优先级 |
| P2 | 黄色 | 中优先级 |
| P3 | 绿色 | 低优先级 |

## 输出结构

```
测试用例
└── 一级模块
    └── 二级模块
        └── 三级模块
            └── TC1: 测试场景 [优先级标记]
                ├── 前置条件: xxx
                └── 测试步骤: xxx
                    └── 预期结果: xxx
```

## 示例

项目包含示例文件 `用户登录测试用例.xlsx`，可以直接运行测试：

```bash
python excel_to_xmind.py
```

生成的文件为 `用户登录测试用例_转换.xmind`，可用XMind 8打开查看。
